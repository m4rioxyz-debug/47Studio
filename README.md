# 47Studio

Welcome to **47Studio**. This repository contains the coming soon landing page for the studio and a built-in Discord webhook bot to broadcast commit details to Discord.

## Features Included

- **Coming Soon Landing Page**: A premium, responsive dark-mode landing page located in the `47` folder, featuring animated gradients, custom styling, and layout.
- **Discord GitHub Webhook Bot**: Integration server using Express and `discord.js` to post push notifications directly to a Discord channel.

## Quick Start

1. Install dependencies:
   ```bash
   npm.cmd install
   ```

2. Copy `.env.example` to `.env` and fill out your Discord bot credentials.

3. Start the server:
   ```bash
   node index.js
   ```

4. Refer to `guide.txt` for step-by-step webhook and Discord setup.
